# galaxy-overkill
 Space shooter game made in Unity



Gameplay:
<br>
Survive endless waves of enemies and boost your spaceship to get the highest score!


<br>
Key bindings:
<br>
Arrow Keys -> Movement

Space Bar -> Shoot

P -> Pause Menu

<br>
<br>
